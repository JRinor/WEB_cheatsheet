Cheatsheet Commandes Linux

Navigation dans le système de fichiers

  • pwd : Affiche le chemin du répertoire courant.
  • [options] [chemin] : Liste le contenu d'un répertoire.
      • Options :
      • -l : Format long.
      • -a : Affiche les fichiers cachés.
      • -h : Tailles lisibles par l'homme.
      • -R : Récursif.
        •
  • cd [chemin] : Change le répertoire courant.
      Raccourcis :
      • cd - : Retourne au répertoire précédent.
      • cd ~ ou cd : Accède au répertoire personnel.
      • cd / : Accède à la racine.
        •

Gestion des fichiers et répertoires

  • touch [options] : Crée un fichier vide ou met à jour l'horodatage.
  • cat [options] : Affiche le contenu d'un fichier.
      Options :
      • -n : Numérote les lignes.
      • -A : Affiche les caractères non imprimables.
  • cp [options] : Copie fichiers et répertoires.
      Options :
      • -n : Numérote les lignes.
      • -A : Affiche les caractères non imprimables.
        •
  • mv [options] : Déplace ou renomme.
      Options :
      • -i : Interactif.
      • -u : Met à jour.
        •
  • rm [options] : Supprime fichiers ou répertoires.
      Options :
      • -r : Récursif.
      • -f : Force la suppression sans confirmation.
      • -i : Interactif.
        •
  • mkdir [options] : Crée un nouveau répertoire.
      Options :
      • -p
        •
  • awk '{print $1}' : Traitement de texte avancé.

Manipulation de texte et recherche

  • grep [options] : Recherche un motif dans un fichier.
      Options :
      • -i : Insensible à la casse.
      • -r : Récursif.
      • -n : Affiche les numéros de ligne.
        •
  • sed 's/ancien/nouveau/g' : Remplace du texte dans un fichier.
  • awk '{print $1}' : Traitement de texte avancé.

Gestion des processus et ressources

  • ps [options] : Affiche les processus en cours.
      Exemple :
      • ps aux
  • top ou htop : Affiche les processus en temps réel.
  • kill [signal] : Envoie un signal à un processus.
  • free -h : Affiche l'utilisation de la mémoire.
  • df -h : Affiche l'utilisation de l'espace disque.

Réseau

  • ifconfig ou ip addr : Affiche la configuration réseau.
  • ping : Teste la connectivité avec un hôte.
  • netstat -tuln : Affiche les connexions réseau actives.

Gestion des utilisateurs et permissions

  • sudo : Exécute une commande avec des privilèges root.
  • chmod [options] : Modifie les permissions d'un fichier.
  • chown [options] : : Change le propriétaire d'un fichier.

Autres commandes utiles

  • man : Affiche le manuel d'une commande.
  • history : Affiche l'historique des commandes.
  • tar [options] : Crée ou extrait des archives.
      Exemple :
      • find /home -name "*.txt" .
